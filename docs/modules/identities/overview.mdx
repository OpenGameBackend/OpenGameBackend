<Tab title="Install">
Run the following command in your project directory:

```bash
opengb module add identities
```

Or add the module `identities` to your `backend.json`.
</Tab>

<Tab title="7 Scripts">
**Public**


- **[List Identities](/modules/identities/scripts/list)** (`list`) List all identities the user is associated with.


**Internal** 


- **[Fetch Identity Data](/modules/identities/scripts/fetch)** (`fetch`) Fetch the data associated with a specific identity for a user.

- **[Link Identity To User](/modules/identities/scripts/link)** (`link`) Link a new identity and its associated data to a user. This is used for login and non-login identities.

- **[Set Identity Data](/modules/identities/scripts/set)** (`set`) Set the data associated with a specific identity for a user.

- **[Sign In or Sign Up With Identity](/modules/identities/scripts/sign_in_or_sign_up)** (`sign_in_or_sign_up`) Sign in to a user with an identity, creating a new user if it fails.

- **[Sign In With Identity](/modules/identities/scripts/sign_in)** (`sign_in`) Sign in to a user with an identity.

- **[Sign Up With Identity](/modules/identities/scripts/sign_up)** (`sign_up`) Sign up with an identity. Creates a new user.

</Tab>
