## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add identities
```

</Tab>

<Tab title="backend.json">

Add the following to your `backend.json`:

```json backend.json
"modules": {
	"identities": {}
}

```
</Tab>

</Tabs>



## Scripts

### Public

These scripts can be called publicly by your game.

<CardGroup><Card title="List Identities" href="/modules/identities/scripts/list">List all identities the user is associated with.</Card></CardGroup>

### Internal

These scripts can only be called by other OpenGB modules.

<CardGroup><Card title="Fetch Identity Data" href="/modules/identities/scripts/fetch">Fetch the data associated with a specific identity for a user.</Card><Card title="Link Identity To User" href="/modules/identities/scripts/link">Link a new identity and its associated data to a user. This is used for login and non-login identities.</Card><Card title="Set Identity Data" href="/modules/identities/scripts/set">Set the data associated with a specific identity for a user.</Card><Card title="Sign In or Sign Up With Identity" href="/modules/identities/scripts/sign_in_or_sign_up">Sign in to a user with an identity, creating a new user if it fails.</Card><Card title="Sign In With Identity" href="/modules/identities/scripts/sign_in">Sign in to a user with an identity.</Card><Card title="Sign Up With Identity" href="/modules/identities/scripts/sign_up">Sign up with an identity. Creates a new user.</Card></CardGroup>

## Errors

- **Identity Provider Already Added To User** (`identity_provider_already_added`) 
- **Identity Provider Already Used By Other User** (`identity_provider_already_used`) 
- **Identity Provider Not Found** (`identity_provider_not_found`)
