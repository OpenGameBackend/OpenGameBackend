No

### Request

_Object containing the following properties:_

| Property             | Type                                                                                               |
| :------------------- | :------------------------------------------------------------------------------------------------- |
| **`userToken`** (\*) | `string`                                                                                           |
| **`info`** (\*)      | _Object with properties:_<ul><li>`identityType`: `string`</li><li>`identityId`: `string`</li></ul> |
| `uniqueData`         | _Object with properties:_<ul></ul>                                                                 |
| `additionalData`     | `unknown` (_nullable_)                                                                             |

_(\*) Required._


## Response



### Response

_Object record with dynamic keys:_

- _keys of type_ `string`
- _values of type_ `never`
