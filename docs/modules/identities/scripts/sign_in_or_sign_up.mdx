---
title: "Sign In or Sign Up With Identity"
description: "Sign in to a user with an identity, creating a new user if it fails."
---

## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.identities.signInOrSignUp({
  // Request body
});
```


</RequestExample>




### Request

_Object containing the following properties:_

| Property         | Type                                                                                               |
| :--------------- | :------------------------------------------------------------------------------------------------- |
| **`info`** (\*)  | _Object with properties:_<ul><li>`identityType`: `string`</li><li>`identityId`: `string`</li></ul> |
| `uniqueData`     | `unknown` (_nullable_)                                                                             |
| `additionalData` | `unknown` (_nullable_)                                                                             |
| `username`       | `string`                                                                                           |

_(\*) Required._


## Response



### Response

_Object containing the following properties:_

| Property             | Type     |
| :------------------- | :------- |
| **`userToken`** (\*) | `string` |
| **`userId`** (\*)    | `string` |

_(\*) Required._
