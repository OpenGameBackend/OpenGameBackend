---
title: "Link Identity To User"
description: "Link a new identity and its associated data to a user. This is used for login and non-login identities."
---

## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.identities.link({
  // Request body
});
```


</RequestExample>




### Request

_Object containing the following properties:_

| Property             | Type                                                                                               |
| :------------------- | :------------------------------------------------------------------------------------------------- |
| **`userToken`** (\*) | `string`                                                                                           |
| **`info`** (\*)      | _Object with properties:_<ul><li>`identityType`: `string`</li><li>`identityId`: `string`</li></ul> |
| `uniqueData`         | `unknown` (_nullable_)                                                                             |
| `additionalData`     | `unknown` (_nullable_)                                                                             |

_(\*) Required._


## Response



### Response

_Object containing the following properties:_

| Property                     | Type                                                                                               |
| :--------------------------- | :------------------------------------------------------------------------------------------------- |
| **`identityProviders`** (\*) | _Array of objects:_<br /><ul><li>`identityType`: `string`</li><li>`identityId`: `string`</li></ul> |

_(\*) Required._
