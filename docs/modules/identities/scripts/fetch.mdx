## Request



### Config

_Object containing the following properties:_

| Property             | Type                                                                                               |
| :------------------- | :------------------------------------------------------------------------------------------------- |
| **`userToken`** (\*) | `string`                                                                                           |
| **`info`** (\*)      | _Object with properties:_<ul><li>`identityType`: `string`</li><li>`identityId`: `string`</li></ul> |

_(\*) Required._


## Response



### Config

_Object containing the following properties:_

| Property        | Type                                                                                                                                          |
| :-------------- | :-------------------------------------------------------------------------------------------------------------------------------------------- |
| **`data`** (\*) | _Object with properties:_<ul><li>`uniqueData`: `unknown` (_nullable_)</li><li>`additionalData`: `unknown` (_nullable_)</li></ul> (_nullable_) |

_(\*) Required._
