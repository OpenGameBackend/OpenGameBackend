---
title: "Fetch Identity Data"
description: "Fetch the data associated with a specific identity for a user."
---

## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.identities.fetch({
  // Request body
});
```


</RequestExample>




### Request

_Object containing the following properties:_

| Property             | Type                                                                                               |
| :------------------- | :------------------------------------------------------------------------------------------------- |
| **`userToken`** (\*) | `string`                                                                                           |
| **`info`** (\*)      | _Object with properties:_<ul><li>`identityType`: `string`</li><li>`identityId`: `string`</li></ul> |

_(\*) Required._


## Response



### Response

_Object containing the following properties:_

| Property        | Type                                                                                                                                          |
| :-------------- | :-------------------------------------------------------------------------------------------------------------------------------------------- |
| **`data`** (\*) | _Object with properties:_<ul><li>`uniqueData`: `unknown` (_nullable_)</li><li>`additionalData`: `unknown` (_nullable_)</li></ul> (_nullable_) |

_(\*) Required._
