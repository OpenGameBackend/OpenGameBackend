## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add currency
```

</Tab>

<Tab title="backend.json">

Add the following to your `backend.json`:

```json backend.json
"modules": {
	"currency": {}
}

```
</Tab>

</Tabs>
