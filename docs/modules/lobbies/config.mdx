title: "Config"
---

This config can be modified by adding the following to your `backend.json`:

```json
{
  "modules": {
    "lobbies": {
      "config": {
        // Your config here
      }
    }
  }
}
```

## Default Value

The provided user config will be merged with this config. If no user config is
  provided, this config will be used.

```json
{
    "lobbies": {
        "regions": [
            "atl",
            "fra"
        ],
        "destroyOnEmptyAfter": 60000,
        "unreadyExpireAfter": 300000,
        "maxPlayers": 16,
        "maxPlayersDirect": 16,
        "enableCreate": false,
        "enableDestroy": false,
        "enableFind": true,
        "enableFindOrCreate": true,
        "enableJoin": true,
        "enableList": true
    },
    "lobbyRules": [],
    "players": {
        "maxPerIp": 8,
        "maxUnconnected": 128,
        "unconnectedExpireAfter": 60000,
        "autoDestroyAfter": 4147200000
    }
}
```
