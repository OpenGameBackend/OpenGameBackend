<Tab title="Install">
Run the following command in your project directory:

```bash
opengb module add auth_email_passwordless
```

Or add the module `auth_email_passwordless` to your `backend.json`.
</Tab>

<Tab title="3 Scripts">
**Public**


- **[Send Email Verification](/modules/auth_email_passwordless/scripts/send_verification)** (`send_verification`) Send a one-time verification code to an email address to verify ownership.

- **[Verify and Add Email to Existing User (No Password)](/modules/auth_email_passwordless/scripts/verify_add_no_pass)** (`verify_add_no_pass`) Verify a user's email address and register it with an existing account. Does not require a password.

- **[Verify and Login as (or Create) User (No Password)](/modules/auth_email_passwordless/scripts/verify_login_or_create_no_pass)** (`verify_login_or_create_no_pass`) Verify the email address code and return a userToken to AN account (creates a new account if one doesn't exist). Does not require a password.

_No internal scripts._
