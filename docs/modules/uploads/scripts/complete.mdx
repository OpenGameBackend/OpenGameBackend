title: "Complete Upload"
description: "Alert the module that the upload has been completed"
---

## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.uploads.complete({
  // Request body
});
```


</RequestExample>




### Request

_Object containing the following properties:_

| Property            | Type     |
| :------------------ | :------- |
| **`uploadId`** (\*) | `string` |

_(\*) Required._


## Response



### Response

_Object containing the following properties:_

| Property          | Type                                                                                                                                                                                                                                                                       |
| :---------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`upload`** (\*) | _Object with properties:_<ul><li>`id`: `string`</li><li>`metadata`: `unknown` (_nullable_)</li><li>`bucket`: `string`</li><li>`contentLength`: `string`</li><li>`createdAt`: `string`</li><li>`updatedAt`: `string`</li><li>`completedAt`: `string` (_nullable_)</li></ul> |

_(\*) Required._
