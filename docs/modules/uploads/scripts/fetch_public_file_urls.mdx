---
title: "Fetch Public File URLs"
description: "Returns public presigned URLs for specified files in an upload"
---

## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.uploads.fetchPublicFileUrls({
  // Request body
});
```


</RequestExample>




### Request

_Object containing the following properties:_

| Property             | Type            |
| :------------------- | :-------------- |
| **`uploadId`** (\*)  | `string`        |
| **`filePaths`** (\*) | `Array<string>` |
| `expirySeconds`      | `number`        |

_(\*) Required._


## Response



### Response

_Object containing the following properties:_

| Property         | Type                                                                                                                                                                                     |
| :--------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`files`** (\*) | _Array of objects:_<br /><ul><li>`uploadId`: `string`</li><li>`url`: `string`</li><li>`path`: `string`</li><li>`mime`: `string` (_nullable_)</li><li>`contentLength`: `string`</li></ul> |

_(\*) Required._
