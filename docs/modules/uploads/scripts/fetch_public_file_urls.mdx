## Request



### Config

_Object containing the following properties:_

| Property             | Type            |
| :------------------- | :-------------- |
| **`uploadId`** (\*)  | `string`        |
| **`filePaths`** (\*) | `Array<string>` |
| `expirySeconds`      | `number`        |

_(\*) Required._


## Response



### Config

_Object containing the following properties:_

| Property         | Type                                                                                                                                                                                     |
| :--------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`files`** (\*) | _Array of objects:_<br /><ul><li>`uploadId`: `string`</li><li>`url`: `string`</li><li>`path`: `string`</li><li>`mime`: `string` (_nullable_)</li><li>`contentLength`: `string`</li></ul> |

_(\*) Required._
