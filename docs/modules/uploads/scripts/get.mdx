## Request

```typescript


### Config

_Object containing the following properties:_

| Property             | Type            |
| :------------------- | :-------------- |
| **`uploadIds`** (\*) | `Array<string>` |
| `includeFiles`       | `boolean`       |

_(\*) Required._

```

## Response

```typescript


### Config

_Object containing the following properties:_

| Property           | Type                                                                                                                                                                                                                                                                                                                                                                                                                                |
| :----------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`uploads`** (\*) | `Array<_Object with properties:_<ul><li>id: string</li><li>metadata: any (_nullable_)</li><li>bucket: string</li><li>contentLength: string</li><li>createdAt: string</li><li>updatedAt: string</li><li>completedAt: string (_nullable_)</li></ul> _and_ _Object with properties:_<ul><li>files: _Array of objects:_<br /><ul><li>path: string</li><li>mime: string (_nullable_)</li><li>contentLength: string</li></ul></li></ul>>` |

_(\*) Required._

```
