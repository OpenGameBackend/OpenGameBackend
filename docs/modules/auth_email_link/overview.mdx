<Tab title="Install">
Run the following command in your project directory:

```bash
opengb module add auth_email_link
```

Or add the module `auth_email_link` to your `backend.json`.
</Tab>

<Tab title="2 Scripts">
**Public**


- **[Send Email Verification](/modules/auth_email_link/scripts/send_verification)** (`send_verification`) Send a one-time verification code to an email address to verify ownership.


**Internal** 


- **[Verify and Link Email Address to User](/modules/auth_email_link/scripts/verify_link_email)** (`verify_link_email`) Verify a user's email address and link it to an existing account without implicitly allowing login.
