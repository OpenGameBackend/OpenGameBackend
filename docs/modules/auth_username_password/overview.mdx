## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add auth_username_password
```

</Tab>

<Tab title="backend.json">

Add the following to your `backend.json`:

```json backend.json
"modules": {
	"auth_username_password": {}
}

```
</Tab>

</Tabs>
