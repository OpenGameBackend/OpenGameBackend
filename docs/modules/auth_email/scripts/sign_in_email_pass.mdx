---
title: "Sign In with Email and Password"
description: "Sign in a user with an email and password."
---

## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.authEmail.signInEmailPass({
  // Request body
});
```


</RequestExample>




### Request

_Object containing the following properties:_

| Property            | Type     |
| :------------------ | :------- |
| **`email`** (\*)    | `string` |
| **`password`** (\*) | `string` |

_(\*) Required._


## Response



### Response

_Object containing the following properties:_

| Property             | Type     |
| :------------------- | :------- |
| **`userToken`** (\*) | `string` |
| **`userId`** (\*)    | `string` |

_(\*) Required._
