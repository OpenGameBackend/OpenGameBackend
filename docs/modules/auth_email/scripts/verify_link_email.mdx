## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.authEmail.verifyLinkEmail({
```

### Request

_Object containing the following properties:_

| Property                     | Type     |
| :--------------------------- | :------- |
| **`verificationToken`** (\*) | `string` |
| **`code`** (\*)              | `string` |
| **`userToken`** (\*)         | `string` |

_(\*) Required._


## Response



### Response

_Object record with dynamic keys:_

- _keys of type_ `string`
- _values of type_ `never`
