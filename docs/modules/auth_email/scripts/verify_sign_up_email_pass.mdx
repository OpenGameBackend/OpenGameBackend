title: "Verify and Sign Up with Email and Password"

## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.authEmail.verifySignUpEmailPass({
  // Request body
});
```


</RequestExample>




### Request

_Object containing the following properties:_

| Property                     | Type     |
| :--------------------------- | :------- |
| **`email`** (\*)             | `string` |
| **`password`** (\*)          | `string` |
| **`verificationToken`** (\*) | `string` |
| **`code`** (\*)              | `string` |

_(\*) Required._


## Response



### Response

_Object containing the following properties:_

| Property             | Type     |
| :------------------- | :------- |
| **`userToken`** (\*) | `string` |

_(\*) Required._
