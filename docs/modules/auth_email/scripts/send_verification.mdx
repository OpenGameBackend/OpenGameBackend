---
title: "Send Email Verification (No Password)"
description: "Send a one-time verification code to an email address to verify ownership. Does not require a password."
---

## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.authEmail.sendVerification({
  // Request body
});
```


</RequestExample>




### Request

_Object containing the following properties:_

| Property         | Type     |
| :--------------- | :------- |
| **`email`** (\*) | `string` |
| `userToken`      | `string` |

_(\*) Required._


## Response



### Response

_Object containing the following properties:_

| Property                | Type                                                         |
| :---------------------- | :----------------------------------------------------------- |
| **`verification`** (\*) | _Object with properties:_<ul><li>`token`: `string`</li></ul> |

_(\*) Required._
