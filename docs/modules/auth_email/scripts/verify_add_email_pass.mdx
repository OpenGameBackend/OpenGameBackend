## Request



### Config

_Object containing the following properties:_

| Property                     | Type                  |
| :--------------------------- | :-------------------- |
| **`userToken`** (\*)         | `string`              |
| **`email`** (\*)             | `string`              |
| **`password`** (\*)          | `string`              |
| **`oldPassword`** (\*)       | `string` (_nullable_) |
| **`verificationToken`** (\*) | `string`              |
| **`code`** (\*)              | `string`              |

_(\*) Required._
