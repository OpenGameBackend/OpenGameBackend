---
title: "Verify and Add Email and Password to existing user"
description: "Verify a user's email address and register it with an existing account. Requires a password."
---

## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.authEmail.verifyAddEmailPass({
  // Request body
});
```


</RequestExample>




### Request

_Object containing the following properties:_

| Property                     | Type                  |
| :--------------------------- | :-------------------- |
| **`userToken`** (\*)         | `string`              |
| **`email`** (\*)             | `string`              |
| **`password`** (\*)          | `string`              |
| **`oldPassword`** (\*)       | `string` (_nullable_) |
| **`verificationToken`** (\*) | `string`              |
| **`code`** (\*)              | `string`              |

_(\*) Required._


## Response



### Response

_Object record with dynamic keys:_

- _keys of type_ `string`
- _values of type_ `never`
