Run the following command in your project directory:

```bash
opengb module add auth_email
```

Or add the module `auth_email` to your `backend.json`.
</Tab>

<Tab title="7 Scripts">
**Public**

_No public scripts._

**Internal** 


- **[Send Email Verification (No Password)](/modules/auth_email/scripts/send_verification)** (`send_verification`) Send a one-time verification code to an email address to verify ownership. Does not require a password.

- **[Sign In with Email and Password](/modules/auth_email/scripts/sign_in_email_pass)** (`sign_in_email_pass`) Sign in a user with an email and password.

- **[Verify and Add Email and Password to existing user](/modules/auth_email/scripts/verify_add_email_pass)** (`verify_add_email_pass`) Verify a user's email address and register it with an existing account. Requires a password.

- **[Verify and Add Email to Existing User (No Password)](/modules/auth_email/scripts/verify_add_no_pass)** (`verify_add_no_pass`) Verify a user's email address and register it with an existing account. Does not require a password.

- **[Verify and Link Email Address to User](/modules/auth_email/scripts/verify_link_email)** (`verify_link_email`) Verify a user's email address and link it to an existing account without allowing login passwordless.

- **[Verify and Login as (or Create) User (No Password)](/modules/auth_email/scripts/verify_login_or_create_no_pass)** (`verify_login_or_create_no_pass`) Verify the email address code and return a userToken to AN account (creates a new account if one doesn't exist). Does not require a password.

- **[Verify and Sign Up with Email and Password](/modules/auth_email/scripts/verify_sign_up_email_pass)** (`verify_sign_up_email_pass`) Sign up a new user with an email and password.

</Tab>
