
# Config



### Config

_Object containing the following properties:_

| Property    | Type     |
| :---------- | :------- |
| `fromEmail` | `string` |
| `fromName`  | `string` |

_All properties are optional._


## Default Config

```json
{
    "fromEmail": "hello@test.com",
    "fromName": "Authentication Code"
}
```
