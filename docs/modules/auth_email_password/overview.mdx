## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add auth_email_password
```

</Tab>

<Tab title="backend.json">

Add the following to your `backend.json`:

```json backend.json
"modules": {
	"auth_email_password": {}
}

```
</Tab>

</Tabs>



## Scripts
