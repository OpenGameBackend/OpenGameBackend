<Tab title="Install">
Run the following command in your project directory:

```bash
opengb module add auth_email_password
```

Or add the module `auth_email_password` to your `backend.json`.
</Tab>

<Tab title="4 Scripts">
**Public**


- **[Send Email Verification](/modules/auth_email_password/scripts/send_verification)** (`send_verification`) Send a one-time verification code to an email address to verify ownership.

- **[Sign In with Email and Password](/modules/auth_email_password/scripts/sign_in_email_pass)** (`sign_in_email_pass`) Sign in a user with an email and password.

- **[Verify and Add Email and Password to existing user](/modules/auth_email_password/scripts/verify_add_email_pass)** (`verify_add_email_pass`) Verify a user's email address and register it with an existing account.

- **[Verify and Sign Up with Email and Password](/modules/auth_email_password/scripts/verify_sign_up_email_pass)** (`verify_sign_up_email_pass`) Verify a user's email address and register a new user with an email and password.
