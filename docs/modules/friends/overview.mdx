<Tab title="Install">
Run the following command in your project directory:

```bash
opengb module add friends
```

Or add the module `friends` to your `backend.json`.
</Tab>

<Tab title="7 Scripts">
**Public**


- **[Accept Request](/modules/friends/scripts/accept_request)** (`accept_request`) Accept a friend request from another user.

- **[Decline Request](/modules/friends/scripts/decline_request)** (`decline_request`) Decline a friend request from another user.

- **[List Friends](/modules/friends/scripts/list_friends)** (`list_friends`) List all friends of a user.

- **[List Incoming Friend Requests](/modules/friends/scripts/list_incoming_friend_requests)** (`list_incoming_friend_requests`) List all friend requests received by a user.

- **[List Outgoing Friend Requests](/modules/friends/scripts/list_outgoing_friend_requests)** (`list_outgoing_friend_requests`) List all friend requests sent by a user.

- **[Remove Friend](/modules/friends/scripts/remove_friend)** (`remove_friend`) Remove a friend from your friends list.

- **[Send Request](/modules/friends/scripts/send_request)** (`send_request`) Send a friend request to another user.


**Internal** 

_No internal scripts._
</Tab>
