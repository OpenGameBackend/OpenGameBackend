## Public

No

## Request


<RequestExample>

```typescript OpenGB Script
const data = await ctx.modules.rivet.call({
  // Request body
});
```

### Request

_Object containing the following properties:_

| Property          | Type               |
| :---------------- | :----------------- |
| **`method`** (\*) | `string`           |
| **`path`** (\*)   | `string`           |
| `body`            | `any` (_nullable_) |

_(\*) Required._


## Response



### Response

_Object containing the following properties:_

| Property | Type               |
| :------- | :----------------- |
| `body`   | `any` (_nullable_) |

_All properties are optional._
