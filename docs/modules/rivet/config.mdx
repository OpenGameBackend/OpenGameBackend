
# Config



### Config

_Object containing the following properties:_

| Property                         | Type     |
| :------------------------------- | :------- |
| **`apiEndpoint`** (\*)           | `string` |
| **`apiEndpointVariable`** (\*)   | `string` |
| **`serviceTokenVariable`** (\*)  | `string` |
| **`gameIdVariable`** (\*)        | `string` |
| **`environmentIdVariable`** (\*) | `string` |

_(\*) Required._


## Default Config

```json
{
    "apiEndpoint": "https://api.rivet.gg",
    "apiEndpointVariable": "RIVET_API_ENDPOINT",
    "serviceTokenVariable": "RIVET_SERVICE_TOKEN",
    "gameIdVariable": "RIVET_GAME_ID",
    "environmentIdVariable": "RIVET_ENVIRONMENT_ID"
}
```
