## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add rivet
```

</Tab>

<Tab title="backend.json">

Add the following to your `backend.json`:

```json backend.json
"modules": {
	"rivet": {
		"config": {
			// Your config here. See below for more details.
		}
	}
}

```
</Tab>

</Tabs>

## Config

### Schema
		
```typescript
export interface Config {
  apiEndpoint?: string;
  serviceTokenVariable?: string;
  [k: string]: any;
}
```

### Default

```json
{
    "apiEndpoint": "https://api.rivet.gg",
    "serviceTokenVariable": "RIVET_SERVICE_TOKEN"
}
```
