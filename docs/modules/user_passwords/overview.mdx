<Tab title="Install">
Run the following command in your project directory:

```bash
opengb module add user_passwords
```

Or add the module `user_passwords` to your `backend.json`.
</Tab>

<Tab title="3 Scripts">
**Public**

_No public scripts._

**Internal** 


- **[Add Password for User](/modules/user_passwords/scripts/add)** (`add`) Register a new userID/password combination. Errors if user already has a password.

- **[Update Password for User](/modules/user_passwords/scripts/update)** (`update`) Update a userID/password combination. Errors if user does not have a password.

- **[Verify Password for User ID](/modules/user_passwords/scripts/verify)** (`verify`) Verify that the provided password matches the provided user ID. Errors on mismatch.

</Tab>


<Tab title="3 Errors">

- **Password is Invalid** (`password_invalid`) 


- **User already has a password** (`user_already_has_password`) 


- **User does not yet have a password** (`user_does_not_have_password`) 

</Tab>
