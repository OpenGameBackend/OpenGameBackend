## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add user_passwords
```

</Tab>

<Tab title="backend.json">

Add the following to your `backend.json`:

```json backend.json
"modules": {
	"user_passwords": {}
}

```
</Tab>

</Tabs>



## Scripts

### Public

These scripts can be called publicly by your game.

<CardGroup>_No public scripts._</CardGroup>

### Internal

These scripts can only be called by other OpenGB modules.

<CardGroup><Card title="Add Password for User" href="/modules/user_passwords/scripts/add">Register a new userID/password combination. Errors if user already has a password.</Card><Card title="Update Password for User" href="/modules/user_passwords/scripts/update">Update a userID/password combination. Errors if user does not have a password.</Card><Card title="Verify Password for User ID" href="/modules/user_passwords/scripts/verify">Verify that the provided password matches the provided user ID. Errors on mismatch.</Card></CardGroup>

## Errors

- **Password is Invalid** (`password_invalid`) 
- **User already has a password** (`user_already_has_password`) 
- **User does not yet have a password** (`user_does_not_have_password`)
