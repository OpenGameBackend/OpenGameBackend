## Installation

<Tabs>

<Tab title="CLI">

```sh CLI
opengb module add users
```

</Tab>

<Tab title="backend.yaml">

Add the following to your `backend.yaml`:

```yaml backend.yaml
modules:
  users:
    config:
      # Your config here. See below for more details.

```

## Errors

- **File Too Large** (`file_too_large`) The file is larger than the configured maximum size for a profile picture
- **Invalid MIME Type** (`invalid_mime_type`) The MIME type for the supposed PFP isn't an image
- **Token Not User Token** (`token_not_user_token`) 
- **Unknown Identity Type** (`unknown_identity_type`)
